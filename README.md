<h2>Installation : </h2>
<p>To install and use the project, follow these steps:</p>
<ol>
    <li>Clone the repository</li>
    <li>Make it available locally on your machine</li>
    <li>Install the dependencies</li>
    <li>Change current directory to "server" directory of the project and run "npm start" in the terminal</li>
    <li>Open the browser and browse to local host on appropriate port number</li>
    <li>You are ready to start working with the project and make changes</li>
</ol>
